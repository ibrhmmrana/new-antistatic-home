# Product Sans / Google Sans Fonts

This directory should contain the following font files:

## Required Files

- `ProductSans-Regular.ttf` (font-weight: 400)
- `ProductSans-Medium.ttf` (font-weight: 500)
- `ProductSans-Bold.ttf` (font-weight: 700)

## Optional Files

- `ProductSans-Italic.ttf` (font-weight: 400, italic)
- `ProductSans-MediumItalic.ttf` (font-weight: 500, italic)
- `ProductSans-BoldItalic.ttf` (font-weight: 700, italic)

## Usage

The fonts are loaded via `@font-face` rules in `app/globals.css` and are available as:
- **"Google Sans"** (primary family name)
- **"Product Sans"** (alias for compatibility)

## Typography Rules

- **Headings (h1-h6)**: Use Google Sans
- **Buttons & CTAs**: Use Google Sans
- **Body text (p, li, input, etc.)**: Use Roboto

See `app/globals.css` for the complete typography rules.
